# Cravings Feature

This module provides tools and games to help users resist cravings and build healthy habits.

## Overview

The cravings feature offers:
- Craving assessment and tracking
- Interactive games to distract from cravings
- Breathing exercises
- Hydration tracking
- Habit-building tools

## Components

### Games (In App.jsx - To be extracted)
Interactive games to distract users during cravings:
- **Reaction Game**: Test reflexes
- **Memory Game**: Pattern matching
- **Focus Game**: Concentration training

### Habits (In App.jsx - To be extracted)
Tools for building healthy replacement habits:
- Water intake tracking
- Exercise logging
- Mindfulness practice

## Craving Assessment

11-point assessment system tracking:
1. Craving intensity
2. Trigger identification
3. Mood state
4. Physical symptoms
5. Confidence to resist
6. Alternative activities
7. Support system access

## Data Tracking

Cravings are logged with:
- Timestamp
- Intensity (1-10)
- Triggers
- Resistance strategies used
- Outcome (resisted/relapsed)
