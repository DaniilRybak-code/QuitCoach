# Progress Feature

This module visualizes user progress through dashboards, charts, and analytics.

## Overview

The progress feature provides:
- Overall progress dashboard
- Streak visualization
- Achievement tracking
- Health improvements metrics
- Financial savings calculator
- Historical data charts

## Components

### ProfileView (In App.jsx - To be extracted)
Main dashboard showing:
- Current streak
- Total days clean
- Achievements unlocked
- Health metrics
- Money saved
- Cravings resisted

### AnalyticsDashboard (`components/AnalyticsDashboard.jsx`)
Advanced analytics dashboard with:
- Trend analysis
- Pattern recognition
- Success predictions
- Insights and recommendations

## Metrics Tracked

1. **Time Metrics**:
   - Current streak
   - Total clean days
   - Longest streak
   - Time since quit date

2. **Health Metrics**:
   - Lung capacity improvement
   - Cardiovascular health
   - Skin health
   - Energy levels

3. **Financial Metrics**:
   - Money saved
   - Pods not consumed
   - Investment potential

4. **Behavioral Metrics**:
   - Cravings resisted
   - Games played
   - Habits built
   - Triggers overcome
